# Contributing

Thank you for helping make InkGen better! This project welcomes issues, feature proposals, and pull requests.

- Read the full guidelines in the repository root: [CONTRIBUTING.md](https://github.com/<your-org>/InkGen/blob/main/CONTRIBUTING.md)
- Review the [Code of Conduct](https://github.com/<your-org>/InkGen/blob/main/CODE_OF_CONDUCT.md)
- Run `ruff check`, `ruff format`, `pytest`, and `mkdocs build --strict` before submitting PRs.

If you are new to the codebase, start with the [Getting Started guide](getting-started.md) and then explore the [Components overview](components/drawing-components.md).
